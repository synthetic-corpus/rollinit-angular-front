What Is this?

This is a tool to automatically roll initative at the push of a button.

This is intended to replace the sometimes cumbersome, and game flow breaking process of players rolling iniative, dm rolling for monsters, then calling out order etc.

This tool still assumes table top play in which the DM will use tents cards. It is optimized for table top, in person play, specifically for Adventure's League games.

How to Prepare Encounters

Prior to your game day, select the encounters button from that nav menu. menu in the upper right.

Each Encounter has a name, and a list of monsters with an initiative motivers. Each "monster" you add can represet a single npc or group. You must enter an initiative modifier between -10 and 10.

For ease of use, you can duplicate the same monster with the copy icon. file_copy

Once saved, your encounters will be saved under the encounters page.

When to add Player Characters

Using the same menu option in the upper right select "player". Here, you can add the players who are at your table. The player forms contains more information to be filled such as AC, DC etc. As always, you must include an initiative modifier between -10 and 10.

Running an Encounter

Navigate to the encounters page from the top menu. Select run your encounter by selected the play button. play_arrow

....And there it is! :-) All monsters, players, have now been rolled and are in initiative order. From here you can arrange your tent cards or use what ever your like to keep track of turns in a round.

Advantage and Disadvantage rolls

All players and NPC start with their inititive as normal. Anywhere you see the dice icon you can set a roll to default as advantage or disadvantage. Colors of dice are color coded.

  • normal roll
  • advantage roll
  • disadvantage roll

At the initiive order screen, you can also click this icon to re-roll dice for individual combtants.

FAQs How are Ties Broken?

Ties are broken like this:

  1. Highest Modifier gets precedence.
  2. If both Modifiers are equal, roll anoter d20
  3. In the extremely unlikely event that there is a tie, modifiers are equal, and the 2nd d20 roll is equal... issue solved by mysteries of the Javascript sort functions
